This is an AI powered morning breifing system. It will fetch the latest news from various sources and generate a summary of the news. It will also fetch the weather information for the user's location and generate a summary of the weather. Finally, it will generate a summary of the news and the weather and generate the output in an audio format and send it on WhatsApp.

## Tech Stack

Python
langchain for orchestration
Ollama for LLM
Postgres for database
Streamlit for UI
